Purpose of program

This program is intended to aid the user in producing subtitle files in the common .srt format. To that end, the GUI allows the user to input text to the various fields in a safe manner: the program performs a lot of checks to ensure that the user is inputting valid data.

Main GUI specification

    File menu:
  • Open -- opens a dialog box which allows the user to select a file. If the user cancels, dialog box is closed with no effect. If the user selects a file which does not conform to the .srt criteria, the dialog box closes with a warning sent to the screen. If the user selects a valid subtitle file, a new File GUI appears, populated with the data from the file. See 'File GUI specification' for more information.
  • New -- creates a new, empty File GUI. See 'File GUI specification' for more information.
  • Exit -- closes all open File GUIs and exits the application. Before closing, it asks the user, for each open File GUI with unsaved changes, whether the file should be saved (equivalent to 'Save' option in File GUI).
    Help menu:
  • Manual -- opens this file or directs user to this file.
  • About -- displays information about the history and creator of this application.

File GUI specification

    File menu:
  • Open -- opens a dialog box which allows the user to select a file. If the user cancels, dialog box is closed with no effect. If the user selects a file which does not conform to the .srt criteria, the dialog box closes with a warning sent to the screen with no other effect. If the user selects a valid subtitle file, the contents of the current File GUI are replaced with data from the newly opened file. If there were any unsaved changes before opening a new file, the user is first asked whether he would like to save those changes.
  • Save -- saves file to path indicated in title bar, or opens a dialog box asking the user for a new location if there is currently no associated filesystem path. Notifies the user of the success or failure of the operation.
  • Save as -- opens a dialog box asking the user for a new save location. Saves the file in .srt format to this location. Notifies the user of the success or failure of the operation.
  • Close -- closes the File GUI and its associated file. If there are unsaved changes, first asks the user whether those changes should be saved. If the File GUI is associated with a path, that path is used, otherwise, 'Save as' is invoked.
